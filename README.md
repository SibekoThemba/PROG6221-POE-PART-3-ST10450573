# PROG6221-POE-PART-3-ST10450573
#Cybersecurity Awareness Chatbot

This a desktop application built with Windows Forms and C# that teaches users about cybersecurity through interactive conversations, task management, and quizzes.


It includes the following features:
Chat Interface
Task Manager
Cybersecurity Quiz
Activity Log
Database Integration
